HTML is the standard foundation used to create and design pages on the World Wide Web. Rather than a programming language that handles complex logic, HTML is a markup language that defines the structure and organization of web content using a system of "tags." These tags act as instructions for web browsers, telling them how to distinguish between different types of informationâ€”such as identifying which text is a primary heading, which is a paragraph, and where an image or a hyperlink should be placed. By nesting these elements within one another, developers create a semantic document that browsers can parse and render into the visual interfaces we interact with every day.

In this exercise I am practicing how to organizes information using lists, links, and images to create a functional layout.
